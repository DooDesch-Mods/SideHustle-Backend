What is Side Hustle?

Side Hustle is a free MelonLoader mod for Schedule I. It adds a public lobby browser to the main menu and lets you host or join public co-op lobbies - bigger than the usual four players, and open to anyone, not just your Steam friends.

When a host has extra mods, joining players get the host's mod set set up for them automatically (from Thunderstore) so a modded session just works. This page only mirrors what's live right now - joining still happens in-game through Steam.

Public lobbies - common questions

How do I find public Schedule I lobbies?

This page lists every open public lobby being hosted with the Side Hustle mod, updated live. If you have Side Hustle installed, the in-game menu shows the same lobbies through Steam.

How do I join a public lobby?

Joining happens in-game. Launch Schedule I with the Side Hustle mod, open the Side Hustle menu and pick a lobby from the join browser. This website is just for seeing what's open right now.

Do I need the same mods as the host?

Only for lobbies marked "Synced only" - there you need the host's exact mod set, and Side Hustle sets it up for you automatically. Vanilla lobbies need no extra mods.

Is Side Hustle free?

Yes. Side Hustle is a free MelonLoader mod for Schedule I - grab it from Thunderstore and it adds a public lobby browser and co-op hosting to the main menu.
